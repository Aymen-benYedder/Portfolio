WordPress Glossary

WordPress
An open-source content management system (CMS) powering over 40% of websites. Built with PHP, it's highly customizable and beginner-friendly.
Theme
A collection of template files and stylesheets that control the appearance and layout of a WordPress website. Themes define design without affecting content.
Plugin
Software that extends WordPress functionality. Plugins add features like contact forms, SEO optimization, e-commerce capabilities, and more without modifying core files.
Gutenberg
The block-based editor introduced in WordPress 5.0. Allows users to create content using blocks instead of shortcodes, providing a more visual editing experience.
WooCommerce
The most popular e-commerce plugin for WordPress. Provides complete e-commerce functionality including product management, shopping cart, and payment processing.
Custom Post Types
Extended content types beyond standard posts and pages. Allow organizing content like portfolios, testimonials, products, or events in custom structures.
Hooks
WordPress functions (actions and filters) allowing developers to modify functionality without editing core files. Hooks enable safe customization and plugin/theme integration.
REST API
WordPress API enabling external applications to interact with WordPress data. Allows building headless WordPress applications and integrating third-party services.
Child Theme
A theme inheriting functionality from a parent theme while allowing safe customization. Updates to the parent theme don't affect your modifications.
Multisite
WordPress network feature managing multiple sites from a single WordPress installation. Ideal for agencies managing multiple client sites or publishing networks.

Frequently Asked Questions

What's the difference between WordPress.com and WordPress.org? ▼
WordPress.com is a hosted solution managed by Automattic with limited customization options. WordPress.org (self-hosted) provides complete control over your site, allowing custom themes, plugins, and full modifications. We recommend WordPress.org for maximum flexibility and control.
Do you build custom themes or use existing ones? ▼
We do both. For simple projects with budget constraints, we customize existing premium themes. For unique branding requirements and specific functionality needs, we build custom themes from scratch using HTML5, CSS3, and WordPress best practices.
Can you migrate my existing site to WordPress? ▼
Yes, we specialize in site migrations from various platforms (HTML, Shopify, Wix, etc.) to WordPress. We handle content migration, URL structure preservation, SEO optimization, and ensure zero downtime during the migration process.
How do you ensure WordPress security? ▼
We implement security best practices including SSL certificates, regular WordPress and plugin updates, strong authentication mechanisms, automated backup strategies, security plugins, and code hardening. We also conduct security audits and provide documentation.
Do you provide WooCommerce setup? ▼
Yes, we provide comprehensive WooCommerce setup including product configuration, payment gateway integration (Stripe, PayPal), shipping and tax configuration, and optimization for sales conversion and user experience.
What's included in maintenance packages? ▼
Our maintenance packages include regular WordPress and plugin updates, security monitoring, automated backup management, performance optimization, content support, and technical assistance. We offer flexible monthly plans tailored to your needs.
Can you optimize my existing WordPress site? ▼
Absolutely. We optimize existing WordPress sites for performance, SEO, and security. Our optimization services include caching setup, image optimization, database optimization, code cleanup, and achieving 100/100 Lighthouse scores.
